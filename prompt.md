قم ببناء موقع شخصي (Portfolio) مع المواصفات التالية:
- النوع: موقع شخصي حديث واحترافي لمطور ويب
- الألوان: اللون الرئيسي #1E90FF (أزرق)، اللون الثانوي #F0F0F0 (رمادي فاتح)، اللون الإضافي #FF6347 (أحمر طماطم)
- الخطوط: العناوين: 'Montserrat'، النصوص: 'Roboto'
- الأقسام:
   1. الهيدر (Header) مع اسم الموقع والشعار
   2. شريط التنقل (Navigation) مع روابط: الرئيسية، عني، المشاريع، تواصل
   3. قسم الـ Hero مع رسالة ترحيبية وزر دعوة لاتخاذ إجراء
   4. قسم عني (About) مع صورة شخصية وسيرة قصيرة
   5. قسم المشاريع (Projects) يحتوي على 3 بطاقات مشاريع (عنوان، صورة، وصف قصير)
   6. قسم المهارات (Skills) مع أشرطة تقدم أو أيقونات
   7. قسم التواصل (Contact) مع نموذج بسيط (الاسم، البريد الإلكتروني، الرسالة)
   8. الفوتر (Footer) مع روابط مواقع التواصل الاجتماعي
- اختياري: شريط جانبي (Sidebar) يحتوي على روابط سريعة أو آخر المنشورات
- مزايا إضافية: تصميم متجاوب (Responsive)، تمرير سلس (Smooth Scrolling)، تأثيرات عند المرور على الأزرار (Hover Effects)
- التركيز على هيكلية واضحة وسهولة الاستخدام ومظهر عصري
